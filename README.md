# Tourism Website

## Overview
This project is a university assignment designed to showcase famous landmarks and tourist attractions from around the world. The website provides visitors with an engaging and educational experience, combining clear information, appealing visuals, and interactive features. It aims to highlight the cultural and historical significance of each landmark while offering a smooth browsing experience.

## Features
- **Multi-page structure**: Each landmark has its own dedicated page with descriptions, images, and interactive elements.
- **Responsive design**: The layout adapts to different screen sizes, ensuring accessibility on desktops, tablets, and mobile devices.
- **Interactive navigation**: Slide-in animations and navigation buttons make exploring the site intuitive and enjoyable.
- **Educational content**: Each page includes concise yet informative text about the landmarkâ€™s history, location, and importance.

## Technologies Used
- **HTML5**: For structuring the content and pages.
- **CSS3**: For styling, responsive layouts, and animations.
- **JavaScript**: For interactivity and dynamic elements.
- **Optional libraries**: (e.g., Bootstrap or Animate.css) to enhance design and user experience.

## Installation & Usage 
1. Open the main `index.html` file in any modern web browser.
2. Use the navigation bar or buttons to explore different landmarks.

## Project Goals
- To provide a student-friendly yet professional website that balances design and functionality.
- To serve as an educational resource for anyone interested in tourism and world landmarks.
- To demonstrate practical skills in web development, design, and documentation.
